## top  

	us ===>> user
	sy ===>> system process
	id ===>> serve sarehaltare
	wa ===>> wait
	load average ===>> 1 - 5 - 15

## inside command  

	d ===>> delay
	z ===>> color
	c ===>> direction
	shift + m ===>> memory
	shift + p ===>> cpu
	1 ===>> all cpu
	h ===>> help
	k ===>> kill
	r ===>> nice

	top -u name user
