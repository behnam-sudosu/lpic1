### send email in local

```bash
mail behnam@localhost -a /(directory) ===>> -a = attch
mail -s "test subject" behnam@localhost ===>> -s = subject
mail -s "test" behnam@localhost < /home/behnam/file1 ===>> read file
mail -s "test" behnam@localhost -a /home/behnam/file1 ===>> -a attach file
mail behnam@localhost < /root/file1 ===>> read file1 and send
mail ===>> show all emails
	? ===>> help
	1 ===>> choose number
	q ===>> quite
	d ===>> delete
/home/milad/mbox ===>> save email
/var/mail/behnam ===>> mailbox
mailq ===>> show email that dosn't send yet  
```
